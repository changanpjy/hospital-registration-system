# 运行指南

## 环境要求

- JDK 8 或以上
- Maven 3.6 或以上
- MySQL 8.x
- Node.js 18 或以上

## 1. 初始化数据库

```bash
cd backend
mysql -uroot -p < 数据库初始化脚本/init.sql
```

脚本会创建 `hospital_registration` 数据库、核心业务表和演示数据。

## 2. 启动后端

确认 `backend/src/main/resources/application.properties` 中的数据库连接正确后执行：

```bash
cd backend
mvn spring-boot:run
```

后端地址：`http://localhost:8081`。

## 3. 启动前端

```bash
cd frontend
npm install
npm run dev
```

前端地址：`http://localhost:5173`。

2. 选择科室。
3. 选择医生。
4. 选择未来 7 天内可预约号源。
5. 选择就诊人。
6. 提交预约，系统生成唯一预约号。
7. 在“我的预约”中查看记录。
8. 预约创建 30 分钟内可取消，超过 30 分钟后后端拒绝取消。
